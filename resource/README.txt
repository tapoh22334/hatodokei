## 本ソフトウェアについて
つくよみちゃんが読み上げるシンプルな時報アプリ。
一時間ごとに時間をお知らせします。

## 操作方法
https://www.youtube.com/watch?v=70C6CVCt1vA

## 公式サイト
https://iwase22334.github.io/hatodokei/

## 機能
- 特定の時報をミュート
- 全体の音量調整
- タスクトレイ
- 設定の自動保存

## 動作環境
Windows / macOS
